# **App Name**: ArduinoVerse

## Core Features:

- Interactive Homepage: Dynamic homepage with categorized Arduino code examples and learning resources.
- Sensor Code Library: Extensive library of code examples for various Arduino sensors.
- Output Device Code Library: Collection of code snippets for Arduino output devices like LEDs, LCDs, and motors.
- Code Editor: Code editor with syntax highlighting for arduino.
- AI Example Suggestion Tool: Suggest relevant code examples and tutorials based on user queries, and incorporate details of past examples to make relevant suggestions.

## Style Guidelines:

- Primary color: Forest green (#388E3C) to evoke nature and electronics.
- Background color: Light green (#E8F5E9), slightly desaturated, to complement the primary green.
- Accent color: Golden yellow (#FFB300) to provide contrast and highlight key interactive elements.
- Headline font: 'Space Grotesk' sans-serif for a modern and techy feel, with 'Inter' for longer blocks of body text.
- Code font: 'Source Code Pro' monospaced for displaying code snippets.
- Use icons to categorize different Arduino sensors, output devices, and tutorials.
- Subtle animations when loading code examples.
